## Mục tiêu

Nắm vững Next.js App Router và các tính năng mới trong Next.js 13+, bao gồm Server Components, Server Actions, và streaming. Mục tiêu là có thể xây dựng được ứng dụng production-ready với performance tối ưu.

## Nội dung chính

### 1. App Router Fundamentals
- File-based routing với app directory
- Layouts và Templates
- Loading UI và Error Handling
- Route Groups và Parallel Routes

### 2. Server Components & Client Components
- Sự khác biệt và khi nào dùng gì
- Data fetching patterns
- Streaming và Suspense
- Component composition best practices

### 3. Server Actions
- Form handling với Server Actions
- Progressive Enhancement
- Revalidation và Cache
- Error handling và validation

### 4. Performance Optimization
- Image optimization
- Font optimization
- Metadata và SEO
- Static và Dynamic rendering
