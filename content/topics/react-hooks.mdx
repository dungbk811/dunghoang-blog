## Mục tiêu

Nắm vững tất cả React Hooks (cả built-in và custom hooks) cùng với các design patterns phổ biến trong React. Mục tiêu là viết code React clean, maintainable và performant.

## Nội dung chính

### 1. Basic Hooks
- **useState**: State management trong functional components
- **useEffect**: Side effects và lifecycle
- **useContext**: Shared state không cần prop drilling

### 2. Advanced Hooks
- **useReducer**: Complex state logic
- **useCallback**: Memoize functions
- **useMemo**: Memoize expensive calculations
- **useRef**: DOM refs và persistent values
- **useLayoutEffect**: Synchronous DOM updates

### 3. Custom Hooks
- Tạo reusable logic với custom hooks
- Best practices khi viết custom hooks
- Common patterns: useLocalStorage, useFetch, useDebounce

### 4. Design Patterns
- Compound Components
- Render Props
- Higher-Order Components (HOC)
- Container/Presentational pattern

## Kết quả đạt được

✅ Hiểu rõ rules of hooks và tránh được các lỗi thường gặp
✅ Biết khi nào dùng hook nào cho performance tối ưu
✅ Tự tin viết custom hooks để tái sử dụng logic
✅ Apply được các design patterns phù hợp với từng tình huống
